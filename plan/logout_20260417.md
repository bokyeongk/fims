# 로그아웃 기능 구현 플랜

## 요구사항 요약
- Sidebar(Desktop) 및 BottomNavigation(Mobile) 가장 하단에 로그아웃 버튼 추가
- 로그아웃 처리: 서버 측 `accessToken` HttpOnly 쿠키 만료 처리
- 로그아웃 완료 후 `/login` 화면으로 이동

---

## API 설계

| Method | Endpoint | 인증 | 설명 |
|--------|----------|------|------|
| POST | `/api/v1/auth/logout` | JWT 쿠키 | `accessToken` 쿠키 즉시 만료(MaxAge=0) 후 204 반환 |

> JWT는 서버에 저장되지 않으므로 블랙리스트 처리 없이 쿠키 삭제만으로 로그아웃 처리한다.

---

## 백엔드 구현

### 변경 파일

| 파일 | 변경 내용 |
|------|----------|
| `AuthController.java` | `POST /api/v1/auth/logout` 엔드포인트 추가 |
| `SecurityConfig.java` | `/api/v1/auth/logout` permitAll 추가 |

### 로그아웃 엔드포인트 동작

```
1. accessToken 쿠키를 MaxAge=0, 동일 Path/HttpOnly 속성으로 덮어쓰기 → 브라우저에서 즉시 삭제
2. HTTP 204 No Content 반환
```

## 프론트엔드 구현

### 신규/변경 파일

| 파일 | 변경 내용 |
|------|----------|
| `src/api/auth.ts` | `useLogout` mutation 훅 신규 생성 |
| `src/components/common/Sidebar.tsx` | 하단에 로그아웃 버튼 추가 |
| `src/components/common/BottomNavigation.tsx` | 하단 탭 바 우측에 로그아웃 버튼 추가 |

### UI 배치

**Sidebar (Desktop)**
```
aside
├── [헤더] FIMS 로고
├── [nav] 메뉴 아이템 목록 (flex-1, 스크롤)
└── [하단 고정] 로그아웃 버튼
     - 아이콘: LogOut (lucide-react)
     - 텍스트: "로그아웃"
     - 스타일: 빨간 계열 텍스트 (text-red-500), 호버 시 bg-red-50
```

**BottomNavigation (Mobile)**
```
탭 바 우측에 로그아웃 아이콘 버튼 추가 (텍스트 없이 아이콘만)
- 아이콘: LogOut (lucide-react)
- 스타일: text-red-400, 호버 시 text-red-600
```

### 로그아웃 플로우

```
[사용자] 로그아웃 버튼 클릭
    → POST /api/v1/auth/logout (withCredentials)
    → 서버: accessToken 쿠키 MaxAge=0 설정
    → 완료 후 window.location.href = '/login' (전체 페이지 리로드로 상태 초기화)
```

> `navigate('/login')` 대신 `window.location.href`를 사용하여 React Query 캐시, 전역 상태 등 클라이언트 메모리를 완전히 초기화한다.

## 작업 순서

1. **backend-architect**: `AuthController`에 logout 엔드포인트 추가, `SecurityConfig` permitAll 추가
2. **frontend-architect**: `useLogout` 훅 생성, Sidebar/BottomNavigation 로그아웃 버튼 추가
